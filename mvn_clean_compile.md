# Maven Build Command

## Command: mvn clean compile

This is a Maven command that performs the following actions:

### Breakdown

- **mvn**: Maven command-line tool
- **clean**: Removes all files generated by the previous build (cleans the target directory)
- **compile**: Compiles the source code of the project

### What It Does

1. **Clean Phase**: Deletes the `target/` directory and all compiled classes, resources, and artifacts from previous builds
2. **Compile Phase**: Compiles all Java source files in `src/main/java/` to bytecode in the `target/classes/` directory

### Usage

```bash
mvn clean compile
```

### When to Use

- After pulling code changes to ensure a fresh build
- Before running tests to ensure source code compiles
- To verify there are no compilation errors in your project
- As part of CI/CD pipeline builds

### Related Commands

- `mvn compile` - Compile without cleaning
- `mvn clean package` - Clean, compile, and create deployable package
- `mvn clean install` - Clean, compile, test, and install to local repository
- `mvn clean test` - Clean and run tests
